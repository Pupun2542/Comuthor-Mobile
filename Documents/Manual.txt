1. clone หรือดาวน์โหลด Respository นี้
2. เปิด text editor ขึ้นมา
3. ใช้คำสั่ง npm install ใน terminal เพื่อลง dependency
4. ใช้คำสั่ง Expo start --dev-client เพื่อรันโปรเจ็ค
5. หลังจากรันเรียบร้อย กด a เพื่อ รันแอพบน emulator
5.1 หากแอพปิดตัวเอง ให้ terminate app ใน emulator และทำข้อ 5 อีกครั้ง
5.2 หากข้อ 5.1 ไม่ได้ให้เปิดแอพจากในมือถือ (หา comuthor) แล้วกดที่ recent project แล้ว terminate app และทำข้อ 5 อีกครั้ง
5.3 หากไม่ได้อีกให้ทำข้อ 5.1 และ 5.2 สลับกันไป หากโชคดีจะทำได้ตั้งแต่ครั้งที่ 2 หากลองแล้วหลายรอบให้กด ctrl c ปิดการทำงาน แล้วใช้คำสั่ง expo run:android หรือทำข้อ 4 อีกครั้ง

ข้อควรระวัง:
1. การใส่รูปภาพจะใส่ได้เฉพาะรูปที่มี metadata เท่านั้น (ใช้รูปถ่ายดีที่สุด)
2. การใส่ลิงค์ลงในช่องกรอกลิงค์ต้องเป็น full url (มี https:// นำหน้า) เท่านั้น
3. เมื่อแก้ไขข้อมูลแล้วให้ย้อนกลับมาที่หน้า group ก่อนแล้วกลับเข้าไปใหม่
